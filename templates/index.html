Kidney CT Scan Classification

Upload a kidney CT scan image. Our CNN model will classify it as Normal or Tumor to support clinical assessment.

Upload CT scan image

Use a kidney CT slice (e.g. JPEG/PNG). The model expects 224×224 input; images are resized automatically.

Prediction result

The model outputs one of two classes: Normal (no tumor) or Tumor. This is for research support only—always confirm with a healthcare provider.

Run a prediction to see the result here.

How it works

This app uses a convolutional neural network (CNN) trained on kidney CT scan images. The pipeline:

  • Resizes your image to 224×224 pixels
  • Runs it through the trained model
  • Returns a class: Normal or Tumor

Ensure you have run training (or pulled the model via DVC) so that artifacts/training/model.h5 exists before using Predict.
